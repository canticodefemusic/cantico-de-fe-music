# Cántico de Fe Music V6.1 Infraestructura

Esta versión mejora la V6.0 y deja la infraestructura más profesional.

## Nuevo en V6.1

- `assets/js/core/store.js`: carga centralizada de datos.
- `assets/js/core/router.js`: rutas separadas.
- `assets/js/services/searchService.js`: búsqueda separada.
- `assets/js/services/contentService.js`: relaciones y mapeo de contenido.
- `components/cards.js`: tarjetas reutilizables.
- Página nueva `Letras`.
- API JSON duplicada en `/api`.
- Service Worker actualizado para V6.1.
- Roadmap y estructura más clara.

## Cloudflare Pages

- Framework preset: None
- Build command: vacío
- Build output directory: /

## Importante

Subir esta versión en la rama `V6`, no en `main`.
