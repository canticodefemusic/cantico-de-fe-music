# Cántico de Fe Music V7.0 Plataforma Completa

Esta es la versión completa/release candidate para probar en la rama `V6`.

## Incluye

- Dashboard general.
- Biblioteca de himnos.
- Letras.
- Playlists.
- Álbumes.
- Videos.
- Devocionales.
- Reproductor global.
- Media Library.
- SEO avanzado.
- Admin completo.
- PWA base.
- Sitemap y robots.
- API JSON local.

## Subir

Sube este ZIP en la rama `V6`, no en `main`.

## Después de subir

Revisa estas páginas:

- `/`
- `/?page=dashboard`
- `/?page=hymns`
- `/?page=player`
- `/?page=media`
- `/?page=seo`
- `/?page=admin`

Cuando todo funcione, podemos preparar el merge final hacia `main`.
