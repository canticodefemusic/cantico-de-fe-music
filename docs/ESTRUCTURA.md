# Estructura V6.1

- `/api`: datos JSON que simulan una API.
- `/assets/data`: copia de seguridad de datos.
- `/assets/js/core`: router y store.
- `/assets/js/services`: lógica de búsqueda y contenido.
- `/components`: piezas reutilizables.
- `/pages`: pantallas principales.
- `/admin`: redirección al panel admin.
- `/pwa`: service worker.
