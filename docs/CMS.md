# Cántico de Fe Music V7.1 CMS Fácil

## Nuevo en V7.1

- Página nueva: `/?page=cms`
- Formulario fácil para crear himnos.
- Vista previa visual.
- Guardar borradores en el navegador.
- Borrar borradores.
- Descargar `hymns.json` completo.
- Copiar JSON solo si lo necesitas.
- Los himnos guardados aparecen en la biblioteca local.
- El Admin técnico ahora recomienda usar CMS Fácil.

## Importante

El navegador no puede escribir directamente en GitHub sin una integración especial.  
Por eso el flujo seguro es:

1. Crear himnos en CMS Fácil.
2. Guardar.
3. Descargar `hymns.json`.
4. Subir/reemplazar ese archivo en GitHub:
   - `/api/hymns.json`
   - `/assets/data/hymns.json`
