# Migration Checklist

Antes de mover archivos:

- [ ] Confirmar que estás en la rama V6.
- [ ] Hacer respaldo del repositorio.
- [ ] Subir este Migration Patch.
- [ ] Ejecutar scan-project.
- [ ] Ejecutar dry-run.
- [ ] Revisar reportes.
- [ ] Confirmar que no se sobrescriben archivos importantes.
- [ ] Migrar manualmente los archivos necesarios.
- [ ] Probar el sitio.
- [ ] Revisar consola del navegador.
- [ ] Hacer commit.

Commit recomendado:

```text
Add V8.0 Migration Patch
```
