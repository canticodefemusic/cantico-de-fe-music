# Cómo instalar este Launch Ready Patch en GitHub

1. Entra a GitHub.
2. Cambia a la rama `V6`.
3. Abre el ZIP.
4. Arrastra las carpetas y archivos a GitHub usando:
   `Add file > Upload files`
5. Confirma que se agreguen:
   - `src/core/launch/`
   - `src/modules/public-pages/`
   - `src/modules/seo/`
   - `src/modules/basic-player/`
   - `src/modules/site-data/`
   - `public/robots.txt`
   - `public/sitemap.xml`
   - `public/manifest.json`
   - `launch-ready.css`
   - `docs/launch/`
6. Commit:
   `Add V8.0 Launch Ready Patch`
