# Cántico de Fe Music V6.4 Media Library

## Nuevo en V6.4

- Página Media Library: `/?page=media`
- Carpeta de portadas: `/assets/img/covers/`
- Carpeta de MP3: `/assets/audio/`
- Carpeta de video: `/assets/video/`
- API nueva: `/api/media.json`
- MediaService para rutas sugeridas y validación.
- Admin con pestaña Media.
- Validación de cover y audioUrl.
- Rutas sugeridas automáticas por ID.

## Reglas de nombres

Usa el ID del himno:

- `fe-que-mueve-montanas.jpg`
- `fe-que-mueve-montanas.mp3`
- `fe-que-mueve-montanas.mp4`

## Cloudflare Pages

- Framework preset: None
- Build command: vacío
- Build output directory: /

## Importante

Subir esta versión en la rama `V6`, no en `main`.
